## Translation Guidelines

### Best Practices
1. **Consistency**: Always use the same Nepali translation for the same English term
2. **Formality**: Use formal/respectful language (तपाईं instead of तिमी)
3. **Context**: Consider the context when translating (e.g., "Apply" can be "आवेदन दिनुहोस्" or "लागू गर्नुहोस्")
4. **Honorifics**: Use appropriate honorific forms (गर्नुहोस्) for actions
5. **Numbers**: Use Devanagari numerals when appropriate (१, २, ३...)

### Common Patterns
- **Action Buttons**: Usually end with "गर्नुहोस्" (do/perform)
- **Navigation**: Use simple, direct translations
- **Descriptions**: Maintain the tone and intent of the original English text
- **Technical Terms**: Some terms like "OTP", "Dashboard" may remain in English or be transliterated

## Usage in Code

```javascript
// Import translation hook
import { useLanguage } from '../hooks/useLanguage'

// In component
const { tPageSync } = useLanguage({ 
  pageName: 'landing', 
  autoLoad: true 
})

// Use translation
const text = tPageSync('hero.title')
// Returns: "Transform Your Career Journey" (EN) or "आफ्नो करियर यात्रा रूपान्तरण गर्नुहोस्" (NE)
```
